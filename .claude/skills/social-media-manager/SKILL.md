# Social Media Manager Agent Skill

Automate social media content creation, posting, and performance tracking.

## When to Use

- Creating social media posts from business updates
- Generating platform-specific content (LinkedIn, FB, IG, Twitter)
- Adding relevant hashtags
- Scheduling posts for optimal times
- Generating performance summaries

## Prerequisites

- Social MCP server configured
- Playwright installed (for auto-posting)
- Social summary generator available

## Quick Start

### Generate Posts from Dashboard

```bash

```

## Platform Guidelines

### LinkedIn
- **Tone:** Professional, business-focused
- **Length:** 150-300 words
- **Hashtags:** 3-5 professional tags
- **Best Time:** Tue-Thu, 9-11 AM

### Facebook
- **Tone:** Friendly, engaging
- **Length:** 100-250 words
- **Hashtags:** 1-3 tags
- **Best Time:** Wed-Fri, 1-3 PM

### Instagram
- **Tone:** Visual, lifestyle
- **Length:** 150-300 words + emojis
- **Hashtags:** 10-15 tags
- **Best Time:** Daily, 6-9 PM

### Twitter
- **Tone:** Concise, timely
- **Length:** < 280 characters
- **Hashtags:** 1-2 tags
- **Best Time:** Daily, multiple times

## Hashtag Strategy

### Business/Professional
`#BusinessGrowth #Entrepreneurship #Success #Leadership #Innovation`

### Motivational
